Theory

Basic of network protocol

Protocol is an established set of rules determine how data is transmitted between different device in the same network. Protocol is a standardize set of rules for formatting and processing data. Network protocol are set of roles convention and data structure that dedicate how device exchange of information in an easy, reliable and secure way.
Primary action of network protocols

  • Communication
  • Network management
  • Security

Communication protocol

Allow different network device to communicate each other.
Common types of communication protocol include:-

  • Automation are used to automate different process in both commercial and personal set such as smart building, cloud technology or self driver vehicle. It is artificial intelligent
  • Instant messaging text based communication an smartphones.
  • Routing permit communication between router and other network device.
  • Bluetooth every bluetooth device including headset,smartphones, computer and etc...
  • File transfer if you have moved files from one device to another either a digital or physical medium you have used file transfer protocol(FTP).
  • Internet protocol allow data to be sent between devices using the internet.

Network management

Protocols define and describe various procedures needed to effectively operate a computer network.
function of network management protocols include:-

  • Connection the protocols establish and maintain stable connection between different device.
  • Linkage aggregation allow you to combine multiple network connection into one link between two devices.
  • Troubleshooting allow network administrator to identify error, affecting the network evaluate the quality of the network connection determine new administrator can fix any issues.

Security(cryptographic protocol)

Work to ensure the network and the data sent over it are protected from unauthorized users.
Common function of securing network protocol

  • Encryption protect data and secure areas by requiring users to input a secret key or password to access information.
  • Entity authentication create a system that regular different devices or user on a network to verify their identify before accessing secure area.
  • Transportation its transported from one network device to other network device.

Common network protocols

  • HTTP(hyper text transfer protocol) this internet protocol defines how data is transmitted our the internet and determine how web servers and browser should respond to the command.
  • SSH(secure socket shell) this protocol provides secure access to a computer even if its on unsecure network.
  • SSM(short message service) this communication protocol was created to send and received text message over cellular network.

Identify network protocol application

Common network protocol application

  • ARP(address resolution protocol) is a communication layer protocol it mapping process between the data link layer and network layer which is used to identify a media access control(MAC) address given the IP address.
  • DNS(domain name system) is a hierarchical system that convert these ip addresses into a human readable host name.
  • FTP(file transfer protocol) is a network protocol on the client and server modes architecture used to transfer files between the client and server on a computer network.
  • IMAP(internet message access protocol) is an internet email protocol that store email on the mail server and allow the end user to retrieve, see and manipulate the message as they were stored locally on the user device. Denial of service(DOS) attack can also be carried out on the email server which result in unrecived and unsent emails.
  • SMB(server message block protocol)
  • RDP(remote desktop protocol) is a protocol that provide users with a geographical interface to connect to other computer over a network connect with when one user runs RDP client software while an other run RDP server software.
  • SMB is a network communication protocol for providing shared access to files,printer and serial ports between nods on a network.
  • SMTP(simple mail transfer protocol) is a communication application layer protocol used to send email and receive email.

Network layer

Network layer is part of internet communication process where these connection occur by sending packet of data between different network.
Function of network layer

  • Breaking up segment into network packets
  • Re assembling the packet on the receiving end
  • Routing packet by discovering the best path across a physical network.

Goal of protocols

  • Used to data tranfer management
  • Error control
  • Used to segmentation and reassembly

Ip address class

The first step in planning for ip addressing on your network is to determine which network class in appropriate for your network. There are classes of tcp/ip network each class uses the 32bit ip addresses space differently.
These class are:-

  • Class A

    Class A network is the first 8 bits of the ip address as it's. Network part the remaining 24bit comprise the host part of the ip address. The value asigned to the first bytes of Class a network number all with the range 0 - 127. All 127 class A network can exist.

  • Class B

    Class B network number uses 16 bits for network number and 16 bits for the host number. The first byte of class B network number is in the range of 128 - 191.

  • Class C

    Class C network number use 24 bit for network number and 8 bit for the host number. The first byte of class C network number is in the range 192 - 223.

Ip address

Ip address is a logical address that is assigned to every single computer that is a part of TCP/IP based network.

Subnet

Subnet is a separate and identifiable portion of an organization network typically arranged for one floor, building or geographical.

Subnet mask

Subnet mask is a 32 bit number used to differentiate the network component of an Ip address by dividing the Ip address.
